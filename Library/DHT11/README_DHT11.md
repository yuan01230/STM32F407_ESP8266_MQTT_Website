# DHT11 温湿度传感器驱动说明

## 1. 简介
本模块提供了 DHT11 温湿度传感器的驱动程序，适用于 STM32F4 系列微控制器。DHT11 是一款含有已校准数字信号输出的温湿度复合传感器，采用专用的数字模块采集技术和温湿度传感技术，确保产品具有极高的可靠性和卓越的长期稳定性。

## 2. 硬件连接
*   **VCC**: 接 3.3V 或 5V 电源。
*   **GND**: 接地。
*   **DATA**: 连接到 MCU 的 GPIO 引脚 (本项目配置为 **PG9**)。
    *   **注意**: DATA 引脚必须接一个 **4.7kΩ ~ 10kΩ** 的上拉电阻，否则无法正常通信。

## 3. 通信协议 (单总线)
DHT11 采用单总线通信协议，一次完整的通信过程约为 4ms。

### 3.1 时序图解
通信过程主要分为三个阶段：
1.  **主机复位信号 (Start Signal)**
2.  **DHT11 响应信号 (Response Signal)**
3.  **数据传输 (Data Transmission)**

#### 阶段一：主机复位 (Start)
*   主机将总线配置为**推挽输出**。
*   主机拉低总线至少 **18ms** (本驱动使用 20ms)，以确保 DHT11 检测到起始信号。
*   主机拉高总线 **20~40us**。
*   主机切换为**输入模式** (上拉)，等待 DHT11 响应。

#### 阶段二：DHT11 响应 (Response)
*   DHT11 检测到起始信号后，发送响应信号：
    *   先拉低总线约 **80us** (表示存在)。
    *   再拉高总线约 **80us** (准备发送数据)。
*   如果主机在规定时间内未检测到低电平或高电平，说明传感器故障或连接断开。

#### 阶段三：数据传输 (Data)
*   DHT11 连续发送 **40位** (5字节) 数据，高位 (MSB) 先行。
*   每一位数据都以 **50us 的低电平时隙** 开始。
*   随后的**高电平持续时间**决定了数据位是 '0' 还是 '1'：
    *   **数据 '0'**: 高电平持续 **26~28us**。
    *   **数据 '1'**: 高电平持续 **70us**。

### 3.2 数据格式 (40 Bits)
一次完整的数据传输包含 40 位数据，分为 5 个字节：

| 字节序号 | 内容 | 说明 |
| :--- | :--- | :--- |
| Byte 0 | 湿度整数部分 | 单位：%RH |
| Byte 1 | 湿度小数部分 | DHT11 此字节通常为 0 |
| Byte 2 | 温度整数部分 | 单位：°C |
| Byte 3 | 温度小数部分 | DHT11 此字节通常为 0 |
| Byte 4 | 校验和 | Byte4 = Byte0 + Byte1 + Byte2 + Byte3 |

**校验规则**: 
如果 `(Byte0 + Byte1 + Byte2 + Byte3) == Byte4`，则数据有效；否则数据校验失败，应丢弃。

## 4. 驱动实现细节 (`dht11.c`)

### 4.1 精准延时
由于 DHT11 对时序要求非常严格 (微秒级)，本驱动使用了基于 DWT (Data Watchpoint and Trace) 的精准延时函数：
*   `delay_us(us)`: 微秒延时
*   `delay_ms(ms)`: 毫秒延时
这确保了在不同主频下都能产生准确的时序。

### 4.2 GPIO 模式切换
驱动中包含两个辅助函数用于快速切换引脚模式：
*   `DHT11_Mode_OUT_PP()`: 切换为推挽输出，用于发送起始信号。
*   `DHT11_Mode_IN()`: 切换为输入模式，用于接收数据。

### 4.3 读取流程
1.  调用 `DHT11_Rst()` 发送起始信号。
2.  调用 `DHT11_Check()` 检测传感器是否存在。
3.  循环 40 次调用 `DHT11_Read_Bit()` 读取每一位数据，通过判断高电平长度区分 '0' 和 '1'。
4.  将读取到的 40 位数据组装成 5 个字节。
5.  计算校验和，如果正确则更新 `DHT11_Data_t` 结构体。

## 5. 如何使用

### 步骤 1: 包含头文件
```c
#include "../../Library/dht11/dht11.h"
```

### 步骤 2: 初始化
在 `main()` 函数的初始化部分调用：
```c
DHT11_Init();
```

### 步骤 3: 读取数据
定义一个数据结构体，并调用读取函数。
**注意**: 由于 DHT11 采样率限制，读取间隔至少为 **1秒 (1000ms)**。

```c
DHT11_Data_t dht_data;

// 在主循环中每隔 1s 调用一次
if (DHT11_ReadData(&dht_data) == 0)
{
    // 读取成功
    printf("湿度: %d.%d %%RH\r\n", dht_data.humi_int, dht_data.humi_dec);
    printf("温度: %d.%d C\r\n", dht_data.temp_int, dht_data.temp_dec);
}
else
{
    // 读取失败 (校验错误或超时)
    printf("DHT11 Error!\r\n");
}
```
