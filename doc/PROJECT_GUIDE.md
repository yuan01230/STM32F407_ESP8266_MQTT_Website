# 项目总览与操作手册

## 1. 项目结构总览

| 模块 | 目录 | 主要职责 | 关键接口 |
| --- | --- | --- | --- |
| UI 页面层 | `Library/ui/` | 主页面、列表页、查看页、按键分发 | `UI_EnterPage`、`UI_LoadDirectory`、`UI_View_Show` |
| 图片预览层 | `Library/Picture/` | `BMP/JPG` 识别、解码、缩放、显示 | `Picture_Show`、`Picture_ShowBMP`、`Picture_ShowJPG` |
| 文件展示辅助 | `Library/SD_RW/` | 文件相关 LCD 菜单展示入口 | `LCD_Show_List_Files_Menu` |
| LCD 驱动层 | `Library/tftlcd/` | LCD 初始化、清屏、绘图、文字显示 | `TFTLCD_Init`、`LCD_Clear`、`LCD_ShowString` |
| 字库存储层 | `Library/font_storage/` | 从 `SD` 导入字库、从外部 Flash 读取字模 | `FontStorage_ImportFromSDEx`、`FontStorage_ReadGlyph16` |
| 编码解析层 | `Library/font_codec/` | 解析 `ASCII / GB2312`，计算字模偏移 | `FontCodec_ParseGB2312` |

## 2. 页面流转关系

| 当前页面 | 触发条件 | 下一页面 | 说明 |
| --- | --- | --- | --- |
| 主页面 `Main` | `KEY0` | 列表页 `List` | 从 `0:/` 根目录开始浏览 |
| 列表页 `List` | 选中目录后按 `KEY2` | 列表页 `List` | 进入子目录并刷新列表 |
| 列表页 `List` | 选中 `[..]` 后按 `KEY2` | 列表页 `List` | 返回上一级目录 |
| 列表页 `List` | 选中文件后按 `KEY2` | 查看页 `View` | 普通文件或图片文件进入查看页 |
| 查看页 `View` | `KEY0` 或 `KEY2` | 列表页 `List` | 返回当前目录列表 |
| 列表页 `List` | `KEY0` | 主页面 `Main` | 退出浏览器 |

## 3. 按键说明

### 3.1 主页面按键

| 按键 | 功能 | 备注 |
| --- | --- | --- |
| `KEY0` | 进入文件浏览器 | 不影响主页面其他功能 |
| `KEY1` | 切换主页面文字颜色 | 演示 UI 刷新效果 |
| `KEY2` | 创建测试文件 | 生成测试文本并显示结果 |
| `KEY_UP` | 重新导入字库 | 从 `SD` 卡重新导入字库 |

### 3.2 列表页按键

| 按键 | 功能 | 备注 |
| --- | --- | --- |
| `KEY1` | 向下选择 | 循环移动选中项 |
| `KEY_UP` | 向上选择 | 循环移动选中项 |
| `KEY2` | 进入目录 / 打开文件 / 返回上一级 | 统一执行“打开当前选中项” |
| `KEY0` | 返回主页面 | 退出文件浏览器 |

### 3.3 查看页按键

| 按键 | 功能 | 备注 |
| --- | --- | --- |
| `KEY0` | 返回列表页 | 图片和普通文件统一行为 |
| `KEY2` | 返回列表页 | 保留双返回键，方便操作 |

## 4. 文件浏览规则

| 项目 | 当前规则 |
| --- | --- |
| 根目录 | 固定从 `0:/` 开始 |
| 子目录进入 | 选中目录后按 `KEY2` |
| 上一级目录 | 非根目录时自动插入 `[..]` |
| 列表项类型 | 父目录项、普通目录、普通文件 |
| 当前目录路径 | 由 `g_file_browse_path` 统一维护 |
| 返回列表后路径 | 保持在原查看文件所在目录 |

## 5. 图片预览规则

| 项目 | `BMP` | `JPG/JPEG` |
| --- | --- | --- |
| 识别方式 | 扩展名 `.bmp` | 扩展名 `.jpg/.jpeg` |
| 解码方式 | 自定义逐行解析 | `TJpgDec` 软件解码 |
| 大图处理 | 最近邻等比缩小 | 利用 `TJpgDec` 离散缩放 |
| 小图处理 | 原尺寸居中 | 原尺寸居中 |
| 输出格式 | 转换为 `RGB565` | 解码后输出到 `RGB565` |

## 6. 图片资源建议

### 6.1 BMP 资源建议

| 项目 | 建议值 |
| --- | --- |
| 文件格式 | `24-bit BMP` |
| 压缩方式 | 无压缩 |
| 推荐用途 | 调试、教学、颜色与方向验证 |

### 6.2 JPG 资源建议

| 项目 | 建议值 |
| --- | --- |
| 文件格式 | `Baseline JPEG` |
| 颜色模式 | `RGB` |
| 推荐尺寸 | `320x240`、`240x320`、`240x240` |
| 不推荐格式 | `Progressive JPEG`、`CMYK JPEG` |

## 7. 图片预览状态说明

| 状态文本 | 说明 | 处理建议 |
| --- | --- | --- |
| `OK` | 图片已成功显示 | 无需处理 |
| `JPG UNSUP` | JPEG 编码方式不受支持 | 重新导出为 `Baseline JPEG` |
| `JPG NOMEM` | JPEG 解码工作区不足 | 降低图片复杂度或继续优化解码工作区 |
| `JPG FORMAT` | JPEG 结构损坏或头错误 | 更换图片文件 |
| `OPEN FAIL` | 文件无法打开 | 检查路径、挂载和文件是否存在 |
| `READ ERR` | 读文件失败 | 检查 SD 卡稳定性 |
| `RANGE ERR` | 预览区域参数非法 | 检查显示区域配置 |

## 9. 开发与维护建议

| 方向 | 建议 |
| --- | --- |
| 调试阶段 | 优先使用 `BMP`，便于定位颜色和方向问题 |
| 资源制作 | `JPG` 统一导出为 `Baseline JPEG` |
| 产品化阶段 | 可进一步转向 `RGB565` 原始资源，提高加载效率 |
| 文档维护 | 保持根 `README.md` 负责总览，模块 `README` 负责细节 |
