# 工业控制舱风格改版设计

**目标**

在不改变现有功能、接口、交互逻辑和页面结构绑定关系的前提下，把阿里云 IoT 仪表盘从偏霓虹太空风改造成更稳重、更专业的工业控制舱风格。

**范围**

- 只调整前端视觉表现
- 保留现有 HTML 结构、元素 ID、类名和 JavaScript 逻辑依赖
- 保留所有设备状态、图表、弹窗、控制组件、日志台和配置流程

**视觉方向**

- 主基调为石墨黑、钢板灰、深海蓝
- 强调工业面板、机柜边框、分区压纹、信号灯
- 重点状态色使用信号青、设备在线绿、告警琥珀、错误红
- 弱化夸张霓虹光晕，改为更克制的金属边缘高光和内部阴影

**界面策略**

- 侧边栏表现为设备舱和系统目录
- 顶部区域强化“控制舱标题 + 当前设备选择 + 系统操作”
- 数据卡片改为监测模块面板
- 中部三区保留原布局，但统一为工业面板质感
- 控制台改为黑底终端窗口风
- 抽屉改为机柜滑出面板风格

**兼容性要求**

- 不修改 `client/app.js` 中依赖的 DOM 查询目标
- 不改接口路径和返回结构
- 不引入新的运行时依赖
- 保持桌面和移动端可用

**验收标准**

- 首页可正常加载
- 设备列表、状态点、图表、控制开关、滑块、弹窗功能不受影响
- 本地仿真和 OpenAPI 在线/离线文案仍按现有逻辑显示
- 页面整体视觉明显转为工业控制舱风格
