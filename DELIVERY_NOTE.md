## 4. 图片资源使用规范

### 4.1 支持情况

| 格式 | 是否支持 | 当前说明 |
| --- | --- | --- |
| `BMP` | 支持 | 推荐 `24-bit`、无压缩 |
| `JPG/JPEG` | 支持 | 推荐 `Baseline JPEG` |
| `PNG` | 暂不支持 | 当前工程未接入运行时解码 |
| `GIF` | 暂不支持 | 当前工程未接入运行时解码 |

### 4.2 推荐资源规格

| 格式 | 推荐规格 | 备注 |
| --- | --- | --- |
| `BMP` | `24-bit`、无压缩、分辨率接近 LCD | 适合调试颜色与方向 |
| `JPG/JPEG` | `Baseline JPEG`、`RGB`、常规导出参数 | 不建议 `Progressive JPEG` |
| 测试尺寸 | `320x240`、`240x320`、`240x240` | 适合当前验证 |

### 4.3 预览状态说明

| 状态文本 | 含义 | 建议处理 |
| --- | --- | --- |
| `OK` | 图片已成功显示 | 无需处理 |
| `JPG UNSUP` | JPEG 编码方式不受支持 | 重新导出为 `Baseline JPEG` |
| `JPG NOMEM` | JPEG 解码工作区不足 | 降低图片复杂度或继续优化解码策略 |
| `JPG FORMAT` | JPEG 文件结构异常 | 更换图片文件 |
| `OPEN FAIL` | 文件无法打开 | 检查路径、文件名和挂载状态 |
| `READ ERR` | 文件读取失败 | 检查 `SD` 卡稳定性 |
| `RANGE ERR` | 预览区域参数异常 | 检查显示区域配置 |
