# STM32F407 SDIO + FATFS 字库与文件浏览演示

本工程基于 `STM32F407`，实现以下联动功能：
- SDIO + FATFS 挂载 SD 卡
- 字库从 SD 导入外部 Flash（EN25Q128）
- 中文/英文混排显示（GB2312 + ASCII）
- 文件列表浏览、文件打开查看、创建测试文件
- 串口日志与 LCD 页面同步提示

## 页面功能总表

| 页面 | 主要显示内容 | 可用按键 | 页面内动作 | 返回规则 |
|---|---|---|---|---|
| 主页面 `Main` | 字体演示、状态信息、按键提示、文件总数 | `KEY0/KEY1/KEY2/KEY_UP` | `KEY0`进列表，`KEY1`切文字色，`KEY2`创建文件，`KEY_UP`导入字库 | 停留主页面，不受30s自动返回影响 |
| 文件列表页 `List` | 文件名/扩展名/大小、箭头选中、倒计时 | `KEY0/KEY1/KEY2/KEY_UP` | `KEY1`下移，`KEY_UP`上移，`KEY2`打开文件，`KEY0`回主页面 | 30s无操作自动回主页面 |
| 文件查看页 `View` | 文件元信息、内容文本或十六进制摘要 | `KEY0/KEY2` | `KEY2`关闭并回列表，`KEY0`直接回主页面 | 30s无操作自动回主页面 |
| 字库导入页 `Import` | 信息区 + 进度区（百分比/字节/进度条） | 由主页面 `KEY_UP` 触发 | 显示导入状态并实时更新 | 导入结束后回主页面 |
| 创建文件结果页 `Create` | 创建结果、RTC时间、写入内容预览 | 由主页面 `KEY2` 触发 | 展示写入结果与内容片段 | 3秒后自动回主页面 |

## 按键行为矩阵

| 按键 | 主页面 Main | 列表页 List | 查看页 View | 导入页 Import |
|---|---|---|---|---|
| `KEY0` | 进入列表页 | 返回主页面 | 返回主页面 | 无 |
| `KEY1` | 切换主页面文字颜色 | 选择下一个文件（循环） | 无 | 无 |
| `KEY2` | 创建 `测试专用.txt` 并显示结果页 | 打开当前选中文件 | 关闭查看并回列表页 | 无 |
| `KEY_UP` | 重新导入字库 | 选择上一个文件（循环） | 无 | 无 |

## 典型操作流程（演示）

| 流程 | 操作步骤 | LCD结果 |
|---|---|---|
| 文件浏览流程 | 主页面按 `KEY0` -> 列表按 `KEY1/KEY_UP` 选择 -> 按 `KEY2` 打开 -> 再按 `KEY2` 关闭 -> 按 `KEY0` 回主页面 | 正常进入/查看/返回，无明显整屏闪烁 |
| 字库重导流程 | 主页面按 `KEY_UP` -> 观察导入进度 -> 导入完成后自动回主页面 | 显示状态、百分比、字节数、进度条 |
| 创建文件流程 | 主页面按 `KEY2` -> 显示创建结果页 -> 3秒后回主页面 | SD中生成 `测试专用.txt`，LCD展示写入结果 |

## 刷新策略（防闪烁）

### 文件列表页
- 首次进入列表页：全量绘制静态框架与列表内容。
- 选择变化时：仅刷新旧选中行和新选中行。
- 翻页时：仅刷新列表区。
- 倒计时变化时：仅刷新倒计时显示区域。

### 字库导入页
- 静态标签（`State/Progress/Bytes`）仅首次绘制。
- 状态值、百分比、字节计数仅在内容变化时局部刷新。
- 进度条仅在百分比变化时更新填充区域。

## 文件显示与编码处理

| 文件内容类型 | 处理方式 | 显示效果 |
|---|---|---|
| `GB2312/ASCII` 文本 | 直接显示 | 正常文本 |
| `UTF-8` 文本 | 尝试转 `GB2312` | 可映射字符正常，无法映射显示 `?` |
| 二进制内容 | 十六进制摘要展示 | 便于调试查看 |

## SD 文件约定

- 字库文件路径：`0:/Font/HZK16`
- 文件浏览范围：`0:/` 根目录文件（不含子目录）
- 测试文件名：`测试专用.txt`

## 目录说明

- `Core/Src/main.c`：页面逻辑、按键调度、UI状态机、文件列表/查看逻辑
- `Library/font_storage/`：字库导入与外部Flash字模读取
- `Library/font_codec/`：GB2312编码与偏移计算
- `Library/tftlcd/`：LCD驱动与混排显示接口
- `Library/key/`：按键扫描
- `FATFS/`：FATFS组件

## 构建

- 硬件：STM32F407开发板 + SD卡 + TFT LCD + 按键
